# 🍕 SQL Project Practice Sheet

This repository contains my personal collection of SQL practice queries and data sets. I am working through these to refine my data analysis skills, focusing on real-world data scenarios.

---

## 📂 Project Structure

This repository includes both the data sources used for analysis and the specific SQL queries created to solve various business problems.

### 📊 Data Sets
* **`pizza_types.csv`**: Contains information about pizza types.
* **`pizzas.csv`**: Contains details regarding specific pizza sizes and pricing.
*  **`orders.csv`**: contains information about orders.
*  **`orders_details csv`**: Contains details regarding specific pizzaid and quantity.
  
### 🔍 SQL Queries
The repository includes solutions for 13 distinct analytical questions:

| Category | Analysis Performed |
| :--- | :--- |
| **Sales** | Order analysis and revenue breakdowns. |
| **Trends** | Cumulative revenue and average pizzas ordered per day. |
| **Categorization** | Category-wise sales and total quantity metrics. |
| **Performance** | Top 3 most ordered pizza types (overall & by category). |

## 🛠 Skills Demonstrated

* **Aggregations & Grouping**: Calculating totals, averages, and revenue trends.
* **Data Analysis**: Determining percentage contributions and filtering top-performing items.
* **Time-Series Logic**: Analyzing orders based on hours of the day.
